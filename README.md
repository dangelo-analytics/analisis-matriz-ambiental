# analisis-matriz-ambiental
Análisis y rediseño de una matriz no estructurada hacia un modelo de datos escalable usando Python y pandas.

# Problema
Este proyecto analiza una matriz de requisitos originalmente diseñada para uso manual, con múltiples inconsistencias 
y estructura no estandarizada.

El dataset presenta:
- Texto libre no estructurado
- Uso intensivo de interpretación humana
- Dificultad para automatización

- # Analisis
Se evaluó la viabilidad de automatización mediante limpieza y transformación con pandas.

# Hallazgo clave
Se concluyó que la estructura original no es escalable ni apta para automatización directa.

# Solucion
Se diseñó un nuevo modelo de datos estructurado, utilizando el dataset original solo como referencia.

# Tecnologias

- Python
- pandas
- Google Colab

 # Conclusion
Este caso demuestra que no todos los datasets deben ser automatizados, y que parte del rol analítico es 
identificar cuándo es necesario rediseñar un sistema desde su base.
